PM Framework Design

To increase startup success rates, it's important to develop management skills and use effective project management approaches. This project is divided into two parts:

  1. Investigating present cases and literature reviews.
  2. Creating a customized project management framework for a Taiwan-based startup.

The case study focuses on the RMML startup team, which faces common challenges encountered by new firms in the field of robotic surgery. By understanding the team's situation and resources, a plan template is provided that helps to define and plan the project. This allows the team to see an overview of the project and ensures everyone is on the same page.

The plan template focuses on two of three project management functions: definition and planning. During the definition stage, issues are framed, which helps to get the team speaking the same language. During the planning stage, details are analyzed by breaking down the activities and linking the relationships of each task. The project manager and team can allocate resources and monitor the schedule based on the plan.

This project is intended for entrepreneurs who are starting new companies and need help running them effectively and efficiently.
